The Wonderful and Wacky Waikato Web Hack (EXTENSION)

More JavaScript!

You can pass information to JavaScript functions and that information can change what happens when the function is run. For example:

function add(x, y) {
  let sum = x + y;
  console.log(sum);
}

This function can take 2 numbers, add them together, then print the sum to the deveoper tools "console". You can run this function by typing the function name "add", followed by an open parenthesis "(", then typing any number you want for the first number, then typing a comma ",", then typing any number you want for the second number, then typing a close parenthesis, then pressing the "enter" key on the keyboard. For example, add(6, 7).

Give it a try!
