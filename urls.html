URL Parameters!

URL stands for Uniform Resource Locator. It tells the browser where to find specific resources (e.g. webpage, image, video). URL parameters are extra pieces of information added to the end of a URL, to pass data to a website. This is very userful for things like creating customised webpage content, filtering, and sorting.

URL parameters appear after a questions mark "?". Each parameter has a name (or key) and a value. The name and value are separated by an equals sign "=". For example,

name=aaron

Multiple parameters are separated by an amphersand "&". For example,

firstname=aaron&lastname=matenga

The base URL points to the resource, then you have a questions mark, then you have the URL parameters. So, a full URL could look something like this

https://amatenga-uni.github.io/web-hack/urls.html?firstname=aaron&lastname=matenga
