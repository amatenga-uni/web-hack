JavaScript!

JavaScript code helps to make a website dynamic and interactive. One way we can add JavaScript code to our website is by putting it inside of an HTML Script element. If you inspect this webpage, you will notice there is a Script element that contains the following code:

function sayHello() {
  alert("Hello!");
}

This means that when the browser loads the page, it will create a function called "sayHello". A function is a reusable block of code that performs a specific task. Whenever the "sayHello" function runs, the browser will pop up an alert box containing the text "Hello!".

You can run this function by opening the developer tools, going to the "console" tab, typing "sayHello()", then pressing the "enter" key on the keyboard. Give it a try!

The "console" tab in the developer tools allows you to write and run your own JavaScript code. In JavaScript, we run a function by writing it's name, followed by open and close parentheses. For example, "sayHello()".
